# Social_Media_Analysis
Social Media Performance Dashboard – Built an interactive Excel dashboard to analyze social media engagement metrics such as likes, shares, comments, views, and engagement rate across multiple platforms. Used Pivot Tables, charts, and KPI indicators to visualize performance trends and generate actionable insights for content strategy.
